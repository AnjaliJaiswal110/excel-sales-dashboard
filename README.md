# Excel Sales Dashboard

This project is an interactive Excel dashboard created using sales data of multiple regions.

## Features
- Region wise filtering
- Sales Executive performance tracking
- Target hit percentage analysis
- Away from target analysis
- Interactive charts and pivot tables

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Analysis
- Conditional Formatting
- Some part of VBA 

## Dataset Columns
- Emp Code
- Sales Executive
- Region
- Day1 – Day5 Sales
- Total Sales
- Target
- Target Hit %
- Away From Target %

## Dashboard Insights
- Compare sales executive performance
- Identify top performing regions
- Monitor target achievement
- Analyze sales distribution visually
